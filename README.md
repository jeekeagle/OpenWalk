# LiteSeed 🌱

够不够轻？有没有核？

这是我在 36 岁给自己的人生算法——以最精简的成本（Lite），封装最高密度的价值内核（Seed）。

摆脱外部评价，追求内在价值。不做疲惫的建筑师，做耐心的播种者。

## 关于

LiteSeed 是一个个人博客与知识沉淀项目，记录关于技术、效率与极简主义的思考。

- **理念**：以轻量级的形式承载高密度的价值
- **内容**：技术实践、认知升级、生活哲学
- **风格**：简洁、直接、有核

## 技术实现

本站点基于 [AstroPaper](https://github.com/satnaing/astro-paper) 主题构建，使用以下技术栈：

- **框架**: [Astro](https://astro.build/)
- **样式**: [TailwindCSS](https://tailwindcss.com/)
- **搜索**: [Pagefind](https://pagefind.app/)
- **部署**: Cloudflare Pages

> 本项目源码：https://github.com/jeekeagle/theo
> 原主题来源：https://github.com/satnaing/astro-paper

## 本地运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev
```

访问 http://localhost:4321 查看效果。

## 常用命令

| 命令 | 说明 |
|------|------|
| `pnpm run dev` | 启动开发服务器 |
| `pnpm run build` | 构建生产版本 |
| `pnpm run preview` | 预览构建结果 |
| `pnpm run format` | 代码格式化 |

## 写作规范

```
文章文件夹/
├── 文章标题.md
└── image.png  # 配图
```

- 图片放在文章末尾
